# Introdução 
 .NET Core Project aim to build an aplication to generate PDF from page

# Para start do Projeto
1. Download SDK .NET core 2.2
2. Run "dotnet restore"
3. Run "dotnet build"
4. Run "dotnet watch run --environment "Development" for localhost.
